# Anadil Gazi Portfolio

Personal portfolio website for **Anadil Gazi**, a **MERN Stack / Full-Stack Web Developer** based in India.

Live project highlights include:
- dark developer-focused single-page design
- responsive layout with mobile navigation
- project showcase with screenshots and links
- downloadable resume
- smooth scrolling and subtle reveal animations

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Sections

- Hero
- Skills
- Projects
- Experience
- Education
- Contact

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Project Structure

```text
app/                App Router pages and layout
components/         UI, layout, and homepage sections
data/               Portfolio content and navigation data
public/             Resume, profile photo, and project images
```

## Customization

Main content is managed from:

- `data/portfolio.ts`
- `data/navigation.ts`
- `public/images/`
- `public/Anadil_FullStack_Developer_Resume.pdf`
